# Data Quality Auditor

AI-powered data quality analysis tool that helps you identify and fix issues in your datasets.

## Features

- **Upload** CSV or Excel files
- **Detect** missing values, outliers, duplicates, and inconsistencies
- **Visualize** data quality issues with charts
- **AI Insights** powered by GPT-4 for actionable recommendations
- **Clean** your data with multiple strategies
- **Download** cleaned data and reports

## Deploy to Vercel

See [DEPLOY.md](./DEPLOY.md) for deployment instructions.

## Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: FastAPI (Python)
- **AI**: OpenAI GPT-4
